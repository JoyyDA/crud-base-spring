# CRUD Base Spring Boot Java


## Sobre o Projeto

CRUD Base Spring Boot é uma aplicação de **exemplo** para gerenciar operações básicas de usuario criar, update, delete (CRUD) utilizando **Spring Boot**, **JPA/Hibernate** e **MySQL**.  
O projeto serve como **base para outros projetos Java web**, fornecendo estrutura organizada para persistência de dados e endpoints REST.

---

## Funcionalidades

- Cadastro de usuários
- Atualização de usuários por ID
- Busca de usuários por email
- Exclusão de usuários por email
- Estrutura pronta para expansão com novas entidades
- Integração com banco de dados MySQL

---

## Tecnologias Utilizadas

- **Java 21**
- **Spring Boot 4**
- **Spring Data JPA**
- **MySQL**
- **Lombok**
- **Maven**

---

## Configure o banco de dados MySQL 
  Para configurar va no application.properties: <br>
  spring.datasource.url=jdbc:mysql://localhost:3306/nome_do_banco <br>
  spring.datasource.username=seu_usuario <br>
  spring.datasource.password=sua_senha <br>
